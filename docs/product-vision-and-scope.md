# Product Vision & Scope: Fast Lease Platform

## 1. Executive Summary

**О проекте**
Fast Lease — это цифровая экосистема для автоматизации полного жизненного цикла аренды автомобилей: от онбординга клиента и скоринга до управления штрафами и ремаркетинга.

**Контекст**
Рынок автолизинга страдает от фрагментированных процессов, ручного документооборота и непрозрачности активов, что ведет к потере выручки и рискам.

**Проблема**
Отсутствие единой связности между операциями, финансами и клиентом создает задержки в сделках, кассовые разрывы из-за неоплаченных штрафов и низкий уровень клиентского сервиса.

### Day in the Life (Ключевые сценарии)

*   **Operations Manager (Операционный менеджер)**
    *   Проверил дашборд на наличие сделок, требующих внимания (SLA < 1 часа).
    *   Изучил загруженные документы клиента и запустил AI-проверку на валидность.
    *   В один клик одобрил сделку, инициировав автоматическую генерацию контракта.
    *   Передал задачу на выдачу авто техническому специалисту через внутренний таск-трекер.

*   **Client (Клиент)**
    *   Получил пуш-уведомление о новом штрафе Salik.
    *   Открыл мобильный портал и оплатил штраф через Apple Pay в два касания.
    *   Запросил продление аренды на месяц через интерфейс самообслуживания.
    *   Скачал обновленный сертификат страховки для поездки.

*   **Finance Officer (Финансист)**
    *   Получил сводный отчет о просроченной дебиторской задолженности.
    *   Настроил агрессивность стратегии Smart Collections для сегмента "High Risk".
    *   Автоматически выставил инвойсы за прошедшие сутки по всем платным дорогам.
    *   Сверил поступления от Stripe с банковской выпиской в реальном времени.

*   **Risk Manager (Риск-менеджер)**
    *   Получил алерт о выезде автомобиля за пределы разрешенной геозоны (Geofence).
    *   Проверил телематические данные и текущую локацию авто на карте.
    *   Связался с клиентом через систему или временно заблокировал зажигание (при наличии интеграции).
    *   Обновил скоринговую модель для данного типа клиентов.

## 2. Product Vision

**Видение**
Создать автономную операционную систему для лизингового бизнеса, где 80% рутинных процессов выполняются алгоритмами, обеспечивая мгновенный сервис для клиентов и полный контроль над активами для бизнеса.

**Ключевые принципы**
*   **Automation-First**: Любое действие, которое повторяется дважды, должно быть автоматизировано (штрафы, инвойсы, напоминания).
*   **Seamless Integration**: Бесшовная интеграция с RTA, Salik, банками и страховщиками — данные движутся без участия человека.
*   **AI-Augmented**: Искусственный интеллект не заменяет людей, а усиливает их решения (скоринг рисков, предиктивное ТО).
*   **Radical Transparency**: Данные доступны в реальном времени всем участникам процесса (в рамках прав доступа).

## 4. High-Level Architecture Overview

Система построена на принципах **Composable Architecture**, обеспечивая гибкость и масштабируемость.

*   **Frontend (User Experience)**
    *   **Tech**: Next.js 14 (App Router), React, Tailwind CSS, Shadcn/UI.
    *   **Role**: Обеспечивает высокопроизводительный, отзывчивый интерфейс для всех порталов (Admin, Ops, Client, Investor).
    *   **AI**: Генеративный UI для подсказок, чат-боты поддержки.

*   **Backend & Data Layer**
    *   **Tech**: Supabase (PostgreSQL), Edge Functions (Deno).
    *   **Role**: Хранение данных, управление аутентификацией (Auth), Row-Level Security (RLS), Real-time подписки.
    *   **AI**: Векторный поиск для базы знаний, логика скоринга.

*   **Integration & Automation Layer**
    *   **Tech**: Supabase Edge Functions, Webhooks, Cron Jobs.
    *   **Role**: Оркестрация бизнес-процессов, связь с внешними системами.
    *   **Integrations**:
        *   *Payments*: Stripe / Network International.
        *   *Govt*: RTA / Salik (через API или парсеры).
        *   *Communication*: Twilio / SendGrid / WhatsApp Business API.
    *   **AI**: OCR для распознавания документов, NLP для анализа коммуникаций в Collections.

*   **IoT & Telematics**
    *   **Tech**: MQTT / HTTP ingestion.
    *   **Role**: Сбор данных с GPS-трекеров автомобилей (пробег, локация, статус двигателя).

## 6. Ключевые инсайты

*   **Скорость — главное преимущество**: Автоматизация проверок и генерации документов позволяет закрывать сделки за часы, а не дни, что критично для конкурентного рынка Дубая.
*   **Штрафы как точка роста**: Автоматизация процессинга штрафов превращает убыточный центр (административные расходы) в источник дополнительной маржинальности (admin fees) и улучшает cash flow.
*   **Клиентский опыт = Удержание**: Self-service портал снижает нагрузку на поддержку на 50% и повышает LTV клиента за счет удобства продления и оплат.
*   **Данные — новая нефть**: Сбор телематики и поведенческих данных клиентов создает базу для внедрения продвинутых AI-моделей скоринга и динамического ценообразования в будущем.
*   **Масштабируемость заложена в архитектуру**: Serverless подход (Supabase + Next.js) позволяет кратно растить нагрузку без перестройки инфраструктуры.
